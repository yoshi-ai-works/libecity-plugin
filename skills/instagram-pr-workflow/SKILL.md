# Instagram PRワークフロースキル

インスタ投稿の「考える」フェーズと「作る」フェーズを明確に2段階に分け、ClaudeとCanva MCPを使って需要調査から投稿完成まで一気通貫で行うスキル。

---

## 全体フロー

```
【STEP 1：分析モード】Claudeと考える
  需要調査（ターゲット別）
    ↓
  需要が低い原因の切り分け（認知・動機・障壁）
    ↓
  ★地元民が気づかない魅力の発掘（web検索 + 外部視点分析）
    ↓
  PR切り口の候補出し（リフレーミング）
    ↓
  投稿の「軸」を1行で決める
    ↓
【STEP 2：制作モード】Canvaで作る
  キャプション・ハッシュタグ生成
    ↓
  Canvaテンプレへの自動流し込み
    ↓
  Canvaリンクを返す → 確認 → 投稿
```

**原則：「何を伝えるか」が決まる前に作り始めない。**
**新原則：「地元民の日常 ＝ 他県民の非日常」を常に意識する。**

---

## STEP 1：分析モード

### 1-1. ユーザーから聞くべき情報

まず以下を確認する（わかる範囲でOK）：

- **対象**：何のPRか（商品・場所・サービス・イベント）
- **アカウントの軸**：どんな発信をしているアカウントか
- **現状の利用者**：誰が使っているか、誰が使っていないか
- **今週のネタ**：何か伝えたいことはあるか
- **テンプレ**：Canvaにテンプレがあるか（ある場合はSTEP2でそれを使う）

情報が不足している場合は質問する。ただし一度に聞きすぎず、最重要の1〜2点に絞る。

---

### 1-2. 需要調査プロンプト

ユーザーの情報をもとに、以下のプロンプトをClaudeに投げて分析する：

```
以下のサービス・場所の需要を、ターゲット別に分析してください。

【対象】{対象}
【現状】{現状の利用者と課題}

ターゲットごとに以下を整理してください：
1. 地元住民 / 既存顧客
2. 新規顧客（県外・観光客など）
3. 若年層・学生
4. （必要に応じて追加セグメント）

各ターゲットについて：
・現在の需要レベル（高/中/低）
・需要が低い場合の根本原因（認知・動機・障壁のどれか）
・需要を高めるための施策案
・インスタで刺さるメッセージの切り口
```

---

### 1-3. 需要が低い原因の切り分け

| 原因 | 内容 | 解決方向 |
|---|---|---|
| **認知の問題** | そもそも知られていない | インスタで存在を見せる |
| **動機の問題** | 知っているけど行く理由がない | リフレーミング（言い換え） |
| **障壁の問題** | やりたいけど最初の一歩が重い | ハードルを下げる施策 |

複数が重なることも多い。ユーザーと一緒に「どれが最大の原因か」を特定する。

---

### 1-4. ★地元民が気づかない魅力の発掘（新機能）

#### なぜ必要か

「地元民の日常 ＝ 他県民の非日常」という構造が、地域PRの最大の盲点。
地元では当たり前すぎて誰も気づかない魅力が、外から見ると圧倒的な差別化になる。

例：島原の「鯉の泳ぐまち」
- 地元民：「いつもある水路の鯉」
- 他県民：「街中の側溝に錦鯉が泳いでる！？透明度が高すぎる！」
  → 水中撮影動画の再生数が異常に伸びる現象が起きている

#### 発掘の手順

**① web検索で外部評価を収集する**

Claudeのweb検索機能を使い、以下の観点で検索する：

```
検索クエリ例：
・「{地名} 観光 感動した」
・「{地名} 旅行 びっくりした」
・「{地名} インスタ 人気」
・「{地名} {スポット名} 再生数 バズ」
・「{地名} 他県民 驚き」
・「{地名} 移住 よかった」
```

**② 外部視点分析プロンプト**

```
以下の地域・場所について、「地元民が当たり前すぎて気づかないが、
他県民・外国人から見ると驚くほど魅力的な要素」を発掘してください。

【地域・場所】{地名・スポット名}
【地元民が普通だと思っていること】{わかれば記入。なければ空欄でOK}

以下の切り口で調査・分析してください：

1. 自然環境の当たり前（水質・空気・景観・地形など）
2. 生活文化の当たり前（食・習慣・街並み・人の温かさなど）
3. 歴史・文化の当たり前（史跡・伝統・ストーリーなど）
4. インフラ・アクセスの当たり前（交通・距離感など）

各項目について：
・地元民の認識（「普通」「当たり前」と思っていること）
・他県民・外国人から見た評価（なぜ驚くか・なぜ魅力的か）
・インスタで刺さる見せ方の切り口
・実際にバズった事例や類似事例（わかれば）

最後に、最も「地元民が気づいていない×他県民に刺さる」要素をTOP3で教えてください。
```

**③ 発掘した魅力を地元民向けに発信するポイント**

地元民に「改めて気づいてもらう」ことが目的。そのための切り口：

| パターン | 例 |
|---|---|
| **数字で驚かせる** | 「1日22万トンの湧水が市内60カ所から。これ、普通じゃないんです。」 |
| **他地域との比較** | 「この透明度、日本でここだけかもしれない。」 |
| **外部の声を借りる** | 「初めて来た人が全員驚く、島原の○○。」 |
| **歴史・背景を語る** | 「1978年に子どもたちのために放流された鯉が、今や観光名所に。」 |
| **日常の再発見** | 「毎日通ってた道が、実は奇跡の風景だったと知ってほしい。」 |

---

### 1-5. リフレーミングプロンプト

需要が低いものをPRするとき、言い換えが最大の武器になる：

```
需要が低いサービスのインスタPR戦略を考えます。

【サービス】{サービス名}
【需要が低い理由（仮説）】{原因}
【ターゲット】{最も優先するターゲット}

以下を提案してください：
1. 「{既存の言い方}」以外の言い換え（リフレーミング）を5案
2. 最初の一歩を下げる施策アイデアを3つ
3. 上記をもとにしたインスタキャプション案を3つ
   （各キャプションに「なぜこの切り口か」の理由も添えること）
```

---

### 1-6. 投稿の「軸」を1行で決める

分析が終わったら、ユーザーと一緒に今回の投稿の軸を1行で決める：

> 例：「地元住民に、サイクリングを『移動』ではなく『朝の非日常体験』として届ける」

この1行が決まってからSTEP2へ進む。

---

## STEP 2：制作モード

### 2-1. 前提確認

Canva MCP連携が必要。未設定の場合はユーザーに案内する：
- **設定方法**：Claude設定 → Connectors → Canva → 接続（3クリック）
- **安定性**：Claude Web版でも使えるが、複雑な操作はClaude Codeが安定

---

### 2-2. キャプション・ハッシュタグ生成プロンプト

STEP1の分析結果をそのまま渡す：

```
以下の条件でインスタ投稿のキャプションを3案作成してください。

【アカウントの軸】{アカウントの方向性}
【今回の投稿の軸】{STEP1で決めた1行}
【ターゲット】{ターゲット}
【今週のネタ】{ネタ・キーワード}
【トーン】{親しみやすい / プロフェッショナル / 地元感 など}

各案に：
・キャプション本文（改行含む、インスタらしい文体）
・ハッシュタグ5〜8個
・「なぜこの切り口にしたか」の理由（1行）
を添えてください。
```

---

### 2-3. Canvaテンプレへの流し込みプロンプト

```
私のCanvaの「{テンプレ名}」を使って、以下の内容でインスタ投稿を作ってください。

【キャプション】{STEP2-2で選んだキャプション}
【ハッシュタグ】{ハッシュタグ}
【画像の雰囲気】{雰囲気・トーン}
【テキスト差替え箇所】{テンプレ内の変更箇所があれば指定}
```

テンプレがない場合は新規作成を依頼：

```
島原のサイクリングロードのインスタ投稿を新規で作ってください。
サイズ：1080×1080px（正方形）
テイスト：{雰囲気}
メインテキスト：{テキスト}
```

---

### 2-4. 完成・確認

- Canvaのリンクが返ってくる
- ユーザーがCanva上で微調整（フォント・色・写真差替えなど）
- 確認後、インスタに投稿

---

## セグメント別・定番PR切り口集

### 地元住民向け（文化が薄い場合）
- 「知ってるようで知らなかった{地名}の○○」
- 「車で通り過ぎていた景色が、自転車だと違って見える」
- 「{曜日}の朝、{時間}だけの特別な時間」

### 観光客向け（認知が低い場合）
- 「{地名}でしかできない体験」
- 「廃線跡・歴史・地形などのストーリー性を前面に」
- 「{地名}に来たら絶対行くべき穴場」

### 学生・若年層向け（UGC促進）
- フォトスポットを作ってハッシュタグを統一
- 「映える景色 × 軽い体験（レンタル・無料など）」
- 投稿を促すキャンペーン的な仕掛け

---

## 注意事項・正直な限界

- プロンプトの質 ＝ 出力の質。情報が多いほど精度が上がる
- 凝ったビジュアル1枚を手作業で作りたい場合は手動の方が早いことも
- Canva MCPはテンプレが決まっている＆量産に最も効果を発揮する
- ブランドキット機能はCanva有料プランが必要
- Claude Web版のMCP連携は複雑な操作が不安定なケースあり（Claude Codeを推奨）

---

## 横展開できる場面

このワークフローは以下すべてに使える：

- 地方観光PR（認知が低いスポット・体験）
- 飲食店のSNS運用（季節メニューのネタ切れ）
- 個人ブランドの発信（何を投稿すればいいかわからない）
- 副業・フリーランスのサービスPR（需要があるかわからない段階）
- 地方自治体・観光協会の定期発信業務
